# VSCode Twig extension

## Download

You can install this extension from VSCode Marketplace - [Modern Twig](https://marketplace.visualstudio.com/items?itemName=Stanislav.vscode-twig)

## Features
- Syntax highlighting
- Diagnostics
- Completion
- Hover
- Signature Help
